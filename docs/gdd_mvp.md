# GDD MVP (1人1周)

## 定位
- 类型：悬疑 / 惊悚 / 鬼神 / 软成人文本氛围
- 核心卖点：5名NPC均可接入AI脑，玩家选择驱动3大趋势结局
- 平台：PC + Web（首周目标为Web可玩版 + Steam商店页）

## 核心循环
1. 进入剧情节点
2. NPC回应（AI或降级台词）
3. 玩家二选一
4. 更新三轴与flags
5. 跳转下一节点
6. 终章根据三轴和关键flags结算

## 三轴数值
- `angelKindness`：善意、保护、公开真相
- `demonMalice`：操控、牺牲、权力导向
- `eldritchCorruption`：黑化、低语、克苏鲁污染

## 内容规模锁定
- NPC：5个
- 章节：3章
- 选择：60个（每章10节点*每节点2选）
- 结局：9个（每趋势3个）

## 约束
- 禁止未成年相关和露骨性暴力内容
- AI失败必须剧情可继续（fallback）
- 本周不做复杂战斗、开放地图与联网多人
